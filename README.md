# PC 프로그램 제작 (SpineAnalyzer)

노트북의 내장 카메라만을 사용하여 사용자의 자세를 추측하고 자세평가를 해주는 프로그램

## 개요

+ 제작 인원: 3명 (종합 설계 프로젝트)
+ 개발 기간 : 약 8개월(2019-03\~2019-06 / 2019-09\~2019-12)
+ 사용 언어: Python, CSS

## 개발 환경

+ Pycharm

## 사용 기술

+ OpenCV (얼굴 인식 AI)
+ PYQT5, CSS (GUI)

## 나의 작업

+ 프로그램 디자인 구성 및 구현
+ 애니메이션과 알림 창을 구현
+ 자세를 수치화
