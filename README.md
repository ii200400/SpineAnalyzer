# SpineAnalyzer
종합설계 프로젝트

제작 인원: 3명 (팀 프로젝트)

개발 환경: Pycharm

사용 기술 : OpenCV(얼굴 인식 AI), PYQT5(GUI)

사용 언어: Python

프로젝트 설명
+ 노트북의 내장 카메라만을 사용하여 사용자의 정면 사진만을 가지고 측면 자세를 추측하는 프로그램
+ 애니메이션과 알림 창을 활용하여 시각적으로, 알람 소리를 울려 청각적으로 자세를 교정할 수 있도록 한다.
+ 자체적으로 자세평가를 하여 자세를 수치화한다.
